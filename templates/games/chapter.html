Now that you have read the paper, it's time to jump on Galaxy to do an analysis by yourself!

From the paper and the work done in level2, you should know that some common steps are:

  • Create a new history.
  • Upload datasets: The datasets in this paper are available on ENA (Project number: PRJEB22200). Upload them to galaxy using urls.
  • Quality control: Assessment of the sequencing quality is crucial to eliminate low-quality reads, adapter contamination, and other sources of errors.
  • Read pre-processing: This involves trimming adapters, removing low-quality reads, filtering out reads with low complexity, and correcting sequencing errors if needed.
  • Taxonomic profiling: This step involves assigning taxonomy to reads, either by aligning reads to a reference database or by using marker genes to classify reads.
  • Statistical analysis: This step involves data normalization, differential abundance analysis, and other statistical methods to identify differences in taxonomic and functional profiles between samples.
  • Data visualization: Visualization of the results can be done using various tools such as heatmaps, bar charts, and network graphs to better understand the microbial community structure and function.
